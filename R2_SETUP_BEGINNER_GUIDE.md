## ステップ2: クレジットカードを登録する

**⚠️ 重要**: R2を使うにはクレジットカード登録が必須です（無料枠でも）。

### 2-1. Billingページへ移動

1. 画面右上の**アカウントアイコン**（丸いアイコン）をクリック
2. メニューから **「Billing」** をクリック

**見つからない場合**:
- 画面左側のメニューを下にスクロール
- 「アカウント」または「Account」セクションを探す

### 2-2. 支払い方法を確認

**「Payment Methods」** というセクションを探します。

**パターンA: すでにカードが登録されている場合**
```
✅ Visa ****1234
   Expires: 12/2026
```
→ このセクションはスキップしてOK！ステップ3へ進んでください。

**パターンB: カードが未登録の場合**
```
⚠️ No payment methods on file
   [Add Payment Method]
```
→ 以下の手順でカードを登録します。

### 2-3. クレジットカード情報を入力

1. **「Add Payment Method」** ボタンをクリック

2. **カード情報を入力**:
   ```
   Card Number: 1234-5678-9012-3456  （あなたのカード番号）
   Expiry Date: MM/YY  （有効期限）
   CVV: 123  （カード裏面の3桁の数字）
   Cardholder Name: YAMADA TARO  （カード名義）
   ```

3. **請求先住所を入力**:
   ```
   Country: Japan
   Zip Code: 100-0001  （郵便番号）
   City: Tokyo  （市区町村）
   Address: 1-1-1 Chiyoda  （住所）
   ```

4. **「Add Payment Method」** ボタンをクリック

### 2-4. 登録完了の確認

画面に以下のメッセージが表示されます：
```
✅ Payment method added successfully
```

または、カード情報が一覧に表示されます：
```
✅ Visa ****1234
   Expires: 12/2026
```

1. 画面左側のメニューから **「R2 Object Storage」** をクリック

### 3-2. R2を有効化

**初めてR2を使う場合**、以下のような画面が表示されます：


## ステップ4: R2バケットを作成する

**「バケット」とは？**  
→ 画像を保存する「フォルダ」のようなものです。

### 4-1. バケット作成を開始

画面右上の **「Create bucket」** ボタンをクリック

### 4-2. バケット名を入力

以下の画面が表示されます：

```
┌────────────────────────────────────┐
│  Create R2 bucket                  │
│                                    │
│  Bucket name *                     │
│  [parts-hub-images            ]   │
│                                    │
│  Location                          │
│  ( ) Automatic (Recommended)       │
│  ( ) Specify region               │
│                                    │
│  Storage class                     │
│  ( ) Standard                      │
│                                    │
│  [Cancel]  [Create bucket]        │
└────────────────────────────────────┘
```

**入力内容**:

1. **Bucket name（バケット名）**: 
   ```
   parts-hub-images
   ```
   ⚠️ **この名前を正確に入力してください**（コピペ推奨）

2. **Location（場所）**:
   - **「Automatic (Recommended)」** を選択（推奨）
   - または何も変更しない

3. **Storage class（ストレージクラス）**:
   - **「Standard」** を選択
   - または何も変更しない

### 4-3. バケットを作成

画面右下の **「Create bucket」** ボタンをクリック

### 4-4. 作成完了の確認

数秒後、以下のような画面に切り替わります：

```
┌────────────────────────────────────┐
│  R2 Buckets                        │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ parts-hub-images             │ │
│  │ 0 objects | 0 B              │ │
│  │ Created: Just now            │ │
│  └──────────────────────────────┘ │
└────────────────────────────────────┘
```

✅ バケット一覧に **「parts-hub-images」** が表示されていればOK！

## ステップ5: カスタムドメインを接続する

**「カスタムドメイン」とは？**  
→ 画像にアクセスするためのURLです。  
例: `https://images.parts-hub-tci.com/photo123.jpg`

### 5-1. バケットの設定画面を開く

1. 作成した **「parts-hub-images」** をクリック

2. 画面上部の **「Settings」** タブをクリック

### 5-2. Public Accessセクションを探す

画面を下にスクロールして、以下のセクションを探します：

```
┌────────────────────────────────────┐
│  Public Access                     │
│                                    │
│  Connect a custom domain to        │
│  allow public access              │
│                                    │
│  [  Connect Domain  ]             │
└────────────────────────────────────┘
```

### 5-3. ドメインを接続

1. **「Connect Domain」** ボタンをクリック

2. 以下の画面が表示されます：

```
┌────────────────────────────────────┐
│  Connect custom domain             │
│                                    │
│  Domain name *                     │
│  [images.parts-hub-tci.com    ]   │
│                                    │
│  [Cancel]  [Connect domain]       │
└────────────────────────────────────┘
```

3. **Domain name（ドメイン名）** に以下を入力:
   ```
   images.parts-hub-tci.com
   ```
   ⚠️ **この名前を正確に入力してください**（コピペ推奨）

4. 画面右下の **「Connect domain」** ボタンをクリック

### 5-4. 接続完了の確認

数秒後、以下のような表示に変わります：

```
┌────────────────────────────────────┐
│  Public Access                     │
│                                    │
│  ✅ images.parts-hub-tci.com       │
│     Status: Active                 │
│                                    │
│  [  Disconnect  ]                 │
└────────────────────────────────────┘
```

✅ **Status: Active** と表示されていればOK！

**⚠️ もし「Pending」と表示されている場合**:
- 問題ありません
- 数分待つと自動的に「Active」に変わります

## ステップ6: Pagesプロジェクトにバケットを接続する

**この作業の目的**:  
→ PARTS HUBのウェブサイトが、作成したバケットに画像を保存できるようにします。

### 6-1. Pagesプロジェクトへ移動

1. 画面左側のメニューから **「Workers & Pages」** をクリック

2. プロジェクト一覧から **「parts-hub」** をクリック

```

│  Projects                          │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ parts-hub                     │ │ ← これをクリック
│  │ Production                    │ │
│  │ Last deployed: 1 hour ago    │ │
│  └──────────────────────────────┘ │
└────────────────────────────────────┘
```

### 6-2. Settings画面を開く

画面上部の **「Settings」** タブをクリック

### 6-3. Functionsセクションを探す

画面を下にスクロールして、以下のセクションを探します：

```
┌────────────────────────────────────┐
│  Functions                         │
│                                    │
│  ▼ R2 bucket bindings             │
│                                    │
│  Variable Name    R2 Bucket       │
│  (empty)                          │
│                                    │
│  [  Add binding  ]                │
└────────────────────────────────────┘
```

### 6-4. R2バインディングを追加

1. **「Add binding」** ボタンをクリック

2. 以下の画面が表示されます：

```
┌────────────────────────────────────┐
│  Add R2 bucket binding             │
│                                    │
│  Production ☑️                     │
│  Preview    ☐                     │
│                                    │
│  Variable name *                   │
│  [R2                          ]   │
│                                    │
│  R2 bucket *                       │
│  [Select bucket ▼            ]   │
│                                    │
│  [Cancel]  [Save]                 │
└────────────────────────────────────┘
```

3. **入力内容**:

   **Production（本番環境）**:
   - ☑️ チェックを入れる（デフォルトでON）

   **Variable name（変数名）**:
   ```
   R2
   ```
   ⚠️ **大文字で「R2」と正確に入力**してください

   **R2 bucket（R2バケット）**:
   - ドロップダウンをクリック
   - **「parts-hub-images」** を選択

4. 画面右下の **「Save」** ボタンをクリック

### 6-5. 追加完了の確認

「Functions」セクションの「R2 bucket bindings」に以下のように表示されます：

```
│  Functions                         │

│                                    │
```

## ❓ よくある質問

### Q1: クレジットカードを登録したくない

**A**: 残念ながら、R2を使うにはクレジットカード登録が必須です。  
ただし、**無料枠（10GB）以内なら料金は発生しません**。

---

### Q2: バケット名を間違えた

**A**: 大丈夫です！以下の手順で削除して作り直せます：

1. R2 → parts-hub-images（間違った名前）
2. Settings → Danger Zone → Delete bucket
3. もう一度「Create bucket」から正しい名前で作成

---

### Q3: カスタムドメインが「Pending」から変わらない

**A**: 最大24時間かかる場合があります。  
通常は5-15分で「Active」に変わります。待っていても大丈夫です。

---

### Q4: バインディング追加で「parts-hub-images」が選択肢に出ない

**A**: バケット作成直後は表示されない場合があります。

**解決方法**:
1. ブラウザを更新（F5キー）
2. 1-2分待ってから再度試す
3. それでもダメなら、一度ログアウトして再ログイン

---

### Q5: 料金はいつ確認できる？

**A**: 毎月月初に請求されます。

**確認方法**:
1. Billing → Invoices
2. 過去の請求書一覧が表示されます
