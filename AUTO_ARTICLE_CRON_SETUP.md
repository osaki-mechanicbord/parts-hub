# 毎日自動コラム生成 設定ガイド

## 📋 概要

このガイドでは、Cloudflare Workers の Cron Triggers を使用して、毎日決まった時間に自動車パーツに関する記事を自動生成・公開する方法を説明します。

---

## 🎯 機能

### 自動実行される内容

1. **トレンドキーワードから自動選択**
   - 35種類以上の人気トピックからランダムに選択
   - 自動車パーツの最新トレンドを反映

2. **記事の自動生成**
   - GPT-4o-mini による2000文字程度の詳細記事
   - 初心者にもわかりやすい実用的な内容

3. **画像の自動生成**
   - DALL-E 3による高品質アイキャッチ画像（1792×1024）
   - 記事内容に合った専門的な画像

4. **自動公開**
   - 生成後すぐに「公開」状態で保存
   - 注目記事として自動設定
   - トップページに即座に表示

---

## 📅 実行タイミング

### デフォルト設定
- **実行時刻**: 毎日 日本時間 午前9:00（UTC 0:00）
- **実行頻度**: 1日1回

### カスタマイズ可能な時刻

Cron式を変更することで、任意の時刻に設定できます：

| 時刻（JST） | Cron式 | 説明 |
|------------|--------|------|
| 午前6:00 | `0 21 * * *` | 早朝配信 |
| 午前9:00 | `0 0 * * *` | 朝の通勤時間 |
| 正午12:00 | `0 3 * * *` | ランチタイム |
| 午後6:00 | `0 9 * * *` | 退勤時間 |
| 午後9:00 | `0 12 * * *` | 就寝前 |

※ JST（日本時間）= UTC + 9時間

## ⚙️ Cloudflare Dashboard での設定手順

### ステップ1: Cloudflare Dashboardにログイン

1. https://dash.cloudflare.com/ にアクセス
2. アカウントにログイン

### ステップ2: プロジェクトを選択

1. 左メニューから **Workers & Pages** を選択
2. **parts-hub** プロジェクトをクリック

### ステップ3: Cron Triggersを設定

1. **Settings** タブを開く
2. 左サイドバーから **Triggers** を選択
3. **Cron Triggers** セクションまでスクロール
4. **Add Cron Trigger** ボタンをクリック

### ステップ4: Cron式を入力

1. **Cron expression** フィールドに以下を入力:
   ```
   0 0 * * *
   ```
   
2. **Route** (呼び出すURL) フィールドに以下を入力:
   ```
   https://parts-hub-tci.com/api/admin/articles/auto-generate-with-image
   ```

3. **Save** ボタンをクリック

### ステップ5: 確認

設定が完了すると、Cron Triggers一覧に表示されます：

```
Cron Expression: 0 0 * * *
Route: https://parts-hub-tci.com/api/admin/articles/auto-generate-with-image
Status: Active
```

## 🛠️ トラブルシューティング

### ❌ Cronが実行されない

**原因1**: Cron Triggerが正しく設定されていない

**解決方法**:
1. Cloudflare Dashboard で Cron Triggers を確認
2. Cron式とRouteが正しいか確認
3. Status が「Active」になっているか確認

**原因2**: OpenAI APIキーが設定されていない

**解決方法**:
1. Cloudflare Dashboard → parts-hub → Settings → Environment variables
2. `OPENAI_API_KEY` が設定されているか確認
3. 未設定の場合は、環境変数を追加

### ❌ 記事生成に失敗する

**原因1**: OpenAI APIキーが無効

**解決方法**:
1. https://platform.openai.com/api-keys でキーを確認
2. キーが「Active」になっているか確認
3. 無効な場合は新しいキーを作成し、環境変数を更新

**原因2**: OpenAI API残高不足

**解決方法**:
1. https://platform.openai.com/account/billing で残高を確認
2. 残高が不足している場合は、チャージを実行
3. 支払い方法が登録されているか確認

**原因3**: APIレート制限

**解決方法**:
1. レート制限に達した場合は、数分待って再試行
2. 使用量を確認: https://platform.openai.com/usage
3. 必要に応じてアカウントをアップグレード

### ❌ 画像が表示されない

**原因**: DALL-E 3生成画像のURLは2時間で有効期限切れ

**解決方法**:
1. 画像URLは最初の2時間のみ有効
2. 記事公開直後に画像が表示されることを確認
3. 長期保存が必要な場合は、Cloudflare R2に画像を保存する実装を追加

## 🔧 高度な設定

### 複数回実行（1日2回など）

Cron Triggerを複数追加:

```
# 朝9:00
0 0 * * *

# 夜9:00
0 12 * * *
```

### 特定の曜日のみ実行

```
# 月曜日のみ
0 0 * * 1

# 月・水・金のみ
0 0 * * 1,3,5

# 平日のみ
0 0 * * 1-5
```

### トピック候補のカスタマイズ

`src/routes/admin.ts` の `topics` 配列を編集:

```typescript
const topics = [
  '独自のトピック1',
  '独自のトピック2',
  // ... 追加のトピック
];
```
