# 🔧 R2バケット名エラーの解決方法

## ❌ 発生したエラー

**バケット名**: `parts-hub-images`  
**エラー**: バケット名に使用できない

---

## 💡 原因

R2のバケット名には以下のルールがあります：
- グローバルで一意（世界中で誰も使っていない名前）
- ハイフン（-）は使えるが、条件がある
- すでに他のアカウントで使われている可能性

おそらく、**すでに誰かが `parts-hub-images` を使用している**ため、使えません。

---

## ✅ 解決方法: 別の名前を使う

以下の候補から選んでください（上から順に推奨）：

### 推奨候補1（最優先）
```
partshubimages
```
- ハイフンなし
- シンプル
- 使える可能性が高い

### 推奨候補2
```
parts-hub-tci-images
```
- ドメイン名の一部を含む
- より一意性が高い

### 推奨候補3
```
partshub2024images
```
- 年号を含める
- ユニークになりやすい

### 推奨候補4
```
tci-parts-images
```
- 会社名（TCI）を前に
- 覚えやすい

### 推奨候補5
```
partshubimg
```
- 短縮形
- タイピングが楽

---

## 📋 作業手順（修正版）

### ステップ1: 新しい名前でバケットを作成

1. R2 Object Storage画面で **「Create bucket」** をクリック

2. **Bucket name** に以下を入力（最初の候補から試す）:
   ```
   partshubimages
   ```

3. **Location**: Automatic

4. **「Create bucket」** をクリック

### ステップ2: エラーが出た場合

**エラーメッセージ例**:
```
❌ Bucket name already exists
❌ Invalid bucket name
```

**対処方法**:
- 上記の候補2を試す: `parts-hub-tci-images`
- それでもダメなら候補3を試す: `partshub2024images`
- というように順番に試す

### ステップ3: 成功したバケット名を記録

成功した名前をメモしてください。例:
```
✅ 使用するバケット名: partshubimages
```

## 💡 バケット名のルール（参考）

R2のバケット名には以下の制限があります：

### 使える文字
- ✅ 小文字の英字（a-z）
- ✅ 数字（0-9）
- ✅ ハイフン（-）※条件付き
- ✅ ピリオド（.）※非推奨

### 使えない文字
- ❌ 大文字（A-Z）
- ❌ アンダースコア（_）
- ❌ スペース
- ❌ 日本語

### その他のルール
- 3文字以上63文字以下
- 先頭と末尾はハイフン不可
- 連続するハイフン不可（--）
- IPアドレス形式は不可（192.168.1.1など）
