```
https://images.parts-hub-tci.com/profiles/{user_id}/{timestamp}.jpg
```

例:
```
https://images.parts-hub-tci.com/profiles/123/1711084800000.jpg
```

### 商品画像
```
https://images.parts-hub-tci.com/products/{product_id}/{timestamp}-{index}.jpg

```

https://images.parts-hub-tci.com/products/456/1711084800000-2.jpg
